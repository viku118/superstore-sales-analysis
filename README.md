# 📊 Superstore Sales Analysis Project

## 🔍 Overview

This project analyzes retail sales data using SQL, Python, and Power BI to identify key business insights and improve decision-making.

---

## 🛠️ Tools Used

* SQL (MySQL)
* Python (Pandas)
* Power BI

---

## 📈 Key Insights

* West region generates the highest sales
* Technology category contributes the most revenue
* Tables and Bookcases are loss-making sub-categories

---

## 📂 Project Structure

* Data Cleaning → Python
* Data Analysis → SQL
* Visualization → Power BI

---

## 📊 Dashboard

Power BI dashboard shows:

* Sales trends
* Regional performance
* Profit and loss analysis

---

## 🚀 Conclusion

This project demonstrates end-to-end data analysis from raw data to actionable insights.
